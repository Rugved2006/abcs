# Project Statement

## Title: Cafe Billing System Using Python

### Objective
The purpose of this project is to design and implement a simple, interactive, and user-friendly **Café Billing System** using Python.  
The system should allow customers to browse menu items, place orders, and automatically generate a detailed bill with all applicable charges.

---

## Problem Statement
In many small cafés and restaurants, billing is often handled manually, which can lead to errors, slower service, and inconsistent calculations of taxes and service charges.  
There is a need for a **digital system** that can:

- Display categorized menus  
- Allow customers to select items easily  
- Calculate totals accurately  
- Apply discounts and taxes automatically  
- Generate a professional receipt  

This project solves the above issues by creating a fully automated billing system.

---

## Scope of the Project

- Display full menu categorized into **Indian**, **Chinese**, and **Italian** cuisines  
- Allow customers to add multiple items with desired quantities  
- Automatically calculate:
  - Subtotal  
  - 10% Discount on orders above ₹3500  
  - 10% Service Charge  
  - 18% GST  
  - Final Grand Total  
- Generate a detailed receipt with all items and charges  
- Works entirely through a Python terminal (no external libraries needed)

---

## Outcome
By using this system, cafés and small restaurants can:

- Reduce manual calculation errors  
- Improve billing speed  
- Provide consistency in pricing  
- Enhance customer experience with professional receipts  

---

## Conclusion
The Café Billing System successfully demonstrates how Python can be used to create real-world, interactive billing applications.  
The project provides a reliable, accurate, and user-friendly method for restaurants to manage customer orders and billing.
